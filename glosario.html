Glosario de Términos

Dato

Un dato es un hecho bruto o un valor elemental que, por sí solo, no tiene significado completo. Los datos pueden ser números, palabras, fechas o símbolos que representan información potencial. Por ejemplo, "25", "Juan" o "01/09/2025" son datos.

Información

La información es el conjunto de datos que han sido procesados, organizados o interpretados para generar significado y utilidad. La información permite tomar decisiones y comprender fenómenos. Por ejemplo, "Juan tiene 25 años" combina datos para generar información relevante.

Sistema

Un sistema es un conjunto de elementos interrelacionados que trabajan juntos para alcanzar un objetivo común. Los sistemas tienen entradas (inputs), procesos y salidas (outputs), y pueden incluir personas, máquinas y procedimientos.

Procesamiento de Datos

El procesamiento de datos es la transformación de datos en información mediante operaciones como recopilación, clasificación, almacenamiento, cálculo y análisis. Por ejemplo, sumar ventas diarias para obtener un total mensual.

Sistemas de Información (SI)

Un sistema de información es un conjunto organizado de componentes (personas, hardware, software, datos y procedimientos) que recolecta, procesa, almacena y distribuye información para apoyar la toma de decisiones, coordinación, análisis y control en una organización.

Chatbot

Sistema de inteligencia artificial que simula conversación humana para resolver consultas de usuarios.

Firebase

Plataforma de desarrollo de aplicaciones móviles y web que ofrece diversos servicios en la nube.

Aplicación nativa

Aplicación desarrollada específicamente para un sistema operativo móvil (iOS o Android).

Roles de usuario

Sistema de permisos que define las acciones disponibles para diferentes tipos de usuarios.
