Nuestro Proyecto

Objetivo General

Desarrollar una aplicación móvil dinámica para la UNET que integre un chatbot académico inteligente y funcionalidades de gestión de información institucional, mejorando la experiencia de toda la comunidad universitaria.

Objetivos Específicos

Autenticación con roles

Sistema seguro con permisos diferenciados para estudiantes, profesores y administrativos

Chatbot académico

Asistente virtual en tiempo real para resolver dudas académicas e institucionales

Repositorio dinámico

Centralización de información académica actualizada y de fácil acceso

Experiencia intuitiva

Interfaz amigable y accesible diseñada para dispositivos móviles

Seguridad y escalabilidad

Infraestructura robusta en Firebase garantizando protección y crecimiento

Misión y Visión

Misión

Ofrecer a la comunidad universitaria una herramienta tecnológica innovadora que facilite el acceso a información académica y administrativa, promoviendo la eficiencia y conectividad en todos los procesos universitarios.

Visión

Convertirse en la aplicación institucional de referencia de la UNET, pionera en integrar inteligencia artificial y gestión académica, siendo reconocida por su contribución a la transformación digital de la universidad.

Planteamiento del Problema

Necesidad Identificada

  • Falta de un medio unificado: No existe una plataforma centralizada para trámites y consultas
  • Comunicación fragmentada: La información se dispersa en múltiples canales no oficiales
  • Información desactualizada: Dificultad para acceder a datos confiables y actualizados
  • Procesos ineﬁcientes: Tiempos prolongados en resolución de trámites y consultas

Características Principales

Chatbot Inteligente

Asistente virtual capacitado para responder preguntas frecuentes sobre trámites, horarios, calificaciones y más.

Gestión Académica

Acceso centralizado a horarios de clase, calendario de evaluaciones y eventos universitarios.

Notificaciones

Sistema de alertas personalizadas para recordatorios importantes y novedades institucionales.

Glosario de Términos

Dato

Un dato es un hecho bruto o un valor elemental que, por sí solo, no tiene significado completo. Los datos pueden ser números, palabras, fechas o símbolos que representan información potencial. Por ejemplo, "25", "Juan" o "01/09/2025" son datos.

Información

La información es el conjunto de datos que han sido procesados, organizados o interpretados para generar significado y utilidad. La información permite tomar decisiones y comprender fenómenos. Por ejemplo, "Juan tiene 25 años" combina datos para generar información relevante.

Sistema

Un sistema es un conjunto de elementos interrelacionados que trabajan juntos para alcanzar un objetivo común. Los sistemas tienen entradas (inputs), procesos y salidas (outputs), y pueden incluir personas, máquinas y procedimientos.

Procesamiento de Datos

El procesamiento de datos es la transformación de datos en información mediante operaciones como recopilación, clasificación, almacenamiento, cálculo y análisis. Por ejemplo, sumar ventas diarias para obtener un total mensual.

Sistemas de Información (SI)

Un sistema de información es un conjunto organizado de componentes (personas, hardware, software, datos y procedimientos) que recolecta, procesa, almacena y distribuye información para apoyar la toma de decisiones, coordinación, análisis y control en una organización.

Chatbot

Sistema de inteligencia artificial que simula conversación humana para resolver consultas de usuarios.

Firebase

Plataforma de desarrollo de aplicaciones móviles y web que ofrece diversos servicios en la nube.

Aplicación nativa

Aplicación desarrollada específicamente para un sistema operativo móvil (iOS o Android).

Roles de usuario

Sistema de permisos que define las acciones disponibles para diferentes tipos de usuarios.
